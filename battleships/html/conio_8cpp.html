Detailed Description

conio implements a partial clone of Borland's console I/O.

Author
Stefan Brandle
Version
0.1
Date
2008 Conio implements the text functionality of Borland's console I/O. That includes cursor positioning, setting foreground/background colors, setting text styles, resetting text attributes, and clearing the screen.
